# Python-class-schedule-checker
A Python program that determines what class period a student is currently in based on the day of the week, current time in military time, and lunch group (A, B, or C). Handles the full weekly schedule including Monday/Tuesday/Friday, Wednesday, and Thursday bell schedules, passing periods, advisory, and personalized lunch times.
School Schedule Checker

A Python program that tells you what class period you are currently in based on the day of the week, current time, and lunch group.
Features

Supports all 5 school days with their unique bell schedules
Handles Monday, Tuesday, and Friday standard schedule
Handles Wednesday and Thursday block schedules
Shows passing periods and advisory
Personalized lunch times for groups A, B, and C

How to use

Run the program in Python
Enter the day of the week
Enter your lunch group (A, B, or C)
Enter the current time in military time (example: 13:05 for 1:05 PM)
The program will tell you what period you are currently in

Example
What day is it? Monday
What Lunch are you? B
What time is it? 11:30
This is during school, 4th Hour.
You have lunch from 11:17 - 11:42.
